### CH5 表格
#### 基本元素
* `<table> ` : 用來標記表格
* `<tr> ` : 用來在表個中標示**一列(ROW)**
* `<td> ` : 用來在一列中標示**儲存格(COL)**
* `<th> ` : 用來在一列中標示**標題**儲存格，其內容會**置中**並加上**粗體**
  ### 表格基本框架
  ```
  <table border="1">
    <tr>
      <th>第一欄</th>
      <th>第二攔</th>
     </tr>
    <tr>
      <td>1-1</td>
      <td>1-2</td>
     </tr>
     <tr>
      <td>2-1</td>
      <td>2-2</td>
     </tr>
     <tr>
      <td>3-1</td>
      <td>3-2</td>
    </tr>
  </table>
  ```
  |第一欄|第二欄|
  |-----|-----|
  | 1-1 | 1-2 |
  | 2-1 | 2-2 |
  | 3-1 | 3-2 |
 
#### 表格標題
* `<caption> ` : 用來指定表格**標題**

#### 合併儲存格
* `rowspan="n"`
* `colspan="n"`


#### 表格的表頭、主體、結尾
* `<thead> ` : 用來標示表格的表頭
* `<tbody> ` : 用來標示表格的主體
* `<tfoot> ` : 用來標示表格的結尾
